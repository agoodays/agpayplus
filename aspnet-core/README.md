# AgPay
AgPay是一套适合互联网企业使用的支付系统，支持多渠道服务商和普通商户模式。已对接微信支付，支付宝，云闪付官方接口，支持聚合码支付。
AgPay基于.Net 8，使用 Web API开发，实现权限管理功能，是一套非常实用的Web开发框架。

前端项目：https://github.com/agoodays/agpayplus/tree/main/ant-design-vue
